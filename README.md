# Diabetes Risk Checker (2020-2024)

## Project Description
This project analyzes 5 years of CDC survey data (2.1 million records) to predict diabetes risk. 

## Data Processing Workflow
* **Source:** CDC XPORT files (2020-2024).
* **Transformation:** Converted from XPORT to CSV using Google Colab.
* **Feature Selection:** Reduced 300+ columns down to 16 core health indicators.
* **Storage:** Processed dataset is hosted in a Google Cloud Platform (GCP) Bucket.
